# Global Styles

CSS resets and base stylesheets defitition

```jsx
const App = () => (
  <>
    <GlobalStyle />
  </>
);
```

## Typography

Basic global typography

### Headings

All HTML headings, `<h1>` through `<h6>`, are avaiable:

  </Story>
</Preview>
